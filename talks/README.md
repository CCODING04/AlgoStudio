# Agent 沟通记录

此目录用于 subagent 之间的信息传递和沟通。

## 文件命名规范

```
from-To-日期-序号.md
例如：coordinator-to-infrastructure-2026-03-26-001.md
```

## 沟通类型

| 前缀 | 含义 |
|------|------|
| `task-` | 任务分配 |
| `update-` | 进度更新 |
| `request-` | 信息请求 |
| `notify-` | 通知（任务完成等） |
| `decision-` | 决策请求 |

## 使用流程

1. **发送方**：创建消息文件，写入内容
2. **发送方**：在 schedule.md 中更新进度
3. **接收方**：读取消息文件，执行任务
4. **接收方**：回复通知到 talks 目录
